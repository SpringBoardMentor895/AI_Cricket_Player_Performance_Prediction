## Current Progress
✅ Data Ingestion  
✅ Data Cleaning  
✅ Feature Engineering (Player-Match level dataset)

## Folder Structure

- data/processed/  -> cleaned + feature engineered datasets
- notebooks/       -> EDA and feature engineering notebooks
- src/             -> reusable python scripts
